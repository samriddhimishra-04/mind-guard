PROJECT OVERVIEW
================
A full-stack mental health platform with Angular 18 frontend and Spring Boot 3.3 
backend. Features patient journaling, AI-powered mood tracking, therapist dashboards,
and intelligent alert escalation.

HOW TO RUN
==========
Backend:
  mvn spring-boot:run -Dspring-boot.run.arguments=--server.port=9000

Frontend:
  npm start
  (Runs on http://localhost:4200)

Test Workflows:
  1. Patient: Register → Login → Create Entry → Get Alert
  2. Therapist: Login → View Patients → Manage Alerts

USER WORKFLOWS
==============
Patient:
  Register → Login → Dashboard → Create Journal Entry → Log Mood → View Alerts

Therapist:
  Login → Dashboard (Therapist Portal) → Search/Assign Patients → View Patient 
  Details → Review Journal Entries → Check Mood Logs → Manage Alerts → Resolve 
  Alerts

  Therapist sees banner → Reviews entry → Resolves alert with notes → 
  Patient notified
