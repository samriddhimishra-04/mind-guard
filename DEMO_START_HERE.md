## 🧠 The 3 LLM Calls Explained (90 seconds)

**When patient writes journal:**

1. **First Call - Sentiment Analysis** 📊
   - Hugging Face analyzes emotional tone
   - Returns: 0.92 = very negative/sad
   - Time: 500-800ms

2. **Second Call - Distress Detection** 🚨
   - Hugging Face finds crisis keywords
   - Returns: 8.2/10 = critical level
   - Time: 400-600ms
   - **Decision:** If > 5.0 → trigger Call #3

3. **Third Call - AI Analysis** 💡
   - Uses results from Call #1 (0.92) and Call #2 (8.2)
   - Generates personalized insights
   - Returns: Text like "Based on sentiment 0.92 and distress 8.2..."
   - Time: 1000-2000ms

**Total time:** ~2-3 seconds, all automatic!
