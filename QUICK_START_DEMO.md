
## Key Points to Explain

### The 3 LLM Calls:

**Call #1: Sentiment Analysis**
- Model: `distilbert-base-uncased-finetuned-sst-2-english`
- Analyzes emotional tone
- Returns: Score 0.0 (negative) to 1.0 (positive)
- Our demo: **0.92 = Very Negative** 😞

**Call #2: Distress Detection**
- Model: Custom distress detection
- Identifies crisis indicators
- Keywords: "anxious", "hopeless", "overwhelmed", "depressed"
- Returns: Score 0-10
- Our demo: **8.2/10 = CRITICAL LEVEL** 🚨

**Call #3: AI Analysis Generation**
- Model: Text generation (Hugging Face)
- Creates personalized insights based on sentiment + distress
- Returns: Helpful recommendations
- Our demo: **Auto-generated support strategies** 💡

### What Happens Behind the Scenes:

1. Patient submits journal entry
2. Backend receives request with JWT token
3. Validates authentication
4. Calls `AiAnalysisService.analyzeEntry()`
5. Simultaneously:
   - Calls Hugging Face API for sentiment
   - Calls Hugging Face API for distress
6. If distress > 5.0:
   - Calls Hugging Face API for AI analysis
   - Creates alert in database
7. Therapist sees alert on dashboard
8. Alert includes AI-generated insights
9. Therapist can take action (contact patient, adjust treatment)

## Troubleshooting

### Issue: Button doesn't work
- **Solution**: Refresh page (F5), clear localStorage

### Issue: Dashboard is blank
- **Solution**: Check browser console (F12), look for API errors

### Issue: Alert doesn't appear
- **Solution**: Wait 2-3 seconds, refresh alerts tab, check backend logs

### Issue: Backend not responding
- **Solution**: 
  ```bash
  curl http://localhost:8081/actuator/health
  ```
  Should return: `{"status":"UP"}`

## Demo Script (Talking Points)

> "Today I'm demonstrating Mind-Guard, an AI-powered mental health platform that shows how machine learning can support patient care.

> **The key innovation is the 3 LLM calls:**

> 1. When a patient creates a journal entry, we immediately analyze their sentiment using a Hugging Face model. This tells us their emotional state.

> 2. Simultaneously, we run a distress detection model to identify if they're in crisis. Keywords like 'hopeless', 'anxious', 'overwhelmed' trigger alerts.

> 3. If the distress level is high, we generate a personalized AI analysis that gives the therapist specific insights about what the patient is experiencing.

> All three calls happen asynchronously - the patient doesn't wait. Within a few seconds, the therapist sees an alert with AI-generated recommendations.

> This demonstrates how LLMs can enhance healthcare by:
> - Reducing manual review burden
> - Identifying high-risk patients automatically
> - Providing data-driven insights
> - Enabling faster intervention"
